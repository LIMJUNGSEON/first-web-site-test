CSS


Cascading Style Sheets (CSS) is a style sheet language used to describe how documents written in HTML or XML
(including SVG and XHTML, which are dialects of XML), are displayed. CSS allows elements to be placed on screen, paper, voice,
or other media. Specify how it should be rendered.
CSS is one of the core languages of the open web, and it has a wide range of W3C specifications. It works as standard for browsers.
In the past, we developed various parts of the CSS specification sequentially, so we developed CSS1, CSS2.1, It was possible to
paste versions like CSS3. However, the official version will not go up to CSS4.
Since CSS3, the range of CSS specifications has expanded enormously, and so has the differences between different CSS modules.
Therefore, it is now more effective to develop and publish recommendations on a module-by-module basis. This is the situation. Now,
instead of uploading versions of CSS specifications, the W3C is periodically taking snapshots of the CSS specification
stable version.
