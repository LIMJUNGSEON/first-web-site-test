HTML

지금나는....무엇을 위해 공부를 하는가?

HTML은 Hypertext Markup Language의 약자로, 웹 페이지의 구조를 정의하는 마크업 언어입니다. <와 >로 묶인 **태그(tag)**를 사용하여 텍스트, 이미지, 링크 등 다양한 콘텐츠를 구조화하며, 웹 브라우저가 이를 해석하여 웹 페이지를 화면에 표시하도록 합니다. 주요 특징 웹의 기본 구성 요소: 웹 페이지의 뼈대를 이루는 가장 기본적인 기술입니다. 하이퍼텍스트 기능: 텍스트 안에 다른 문서로 이동할 수 있는 하이퍼링크를 포함할 수 있습니다. 구조 정의: 텍스트의 단락, 목록, 제목 등을 구조화하며, HTML은 웹 페이지의 의미와 구조를 정의합니다. 태그와 요소: HTML 문서는 <태그>콘텐츠 형태로 구성된 **요소(element)**들의 집합입니다. 콘텐츠가 없는 태그는 와 같이 표현됩니다. 다른 기술과의 연계: 웹 페이지의 모양을 꾸미는 **CSS(Cascading Style Sheets)**나 웹 페이지의 동적인 기능을 구현하는 JavaScript와 함께 사용됩니다.
