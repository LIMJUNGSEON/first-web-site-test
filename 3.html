JavaScript


JavaScript (or "JS") is a programming language used to dynamically handle web pages on the client side,
but it is often used on the server side using a runtime such as Node.js.
JavaScript should not be confused with the Java programming language. "Java" and "JavaScript" are trademarks
(or registered trademarks) of Oracle in the United States and other countries, but the two programming languages differ significantly
in their syntax, semantics, and use cases.
Most often, JavaScript is used in browsers. Developers can manipulate the content of web pages through the DOM and handle
the data with AJAX and IndexedDB. You can draw with canvas and interact with devices running browsers using various APIs.
With the improvement and growth of browser APIs, JavaScript has become one of the most spoken languages in the world.
